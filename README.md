# tiny-curl-ios

A simple script to compile tiny-curl to a dylib framework for iOS apps.

# Instructions

It's as simple as:

```
./build-ios.sh <tiny-curl version>
```

Then add the resulting `curl.framework` package to your app and you're finished.
